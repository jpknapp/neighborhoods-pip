To run the program via command line (Unix, OSX)

./neighborhoods.rkt [neighborhoods file] [test points file]

ex:
./neighborhoods.rkt ./gr_neighborhoods.txt ./sample_points.txt

or using racket:
racket neighborhoods.rkt ./gr_neighborhoods.txt ./sample_points.txt

Arguments:
1: neighborhood definitions. this must be a txt file in the format supplied. must have an empty line after every polygon

2: test points. this must be a txt file in the format supplied.
